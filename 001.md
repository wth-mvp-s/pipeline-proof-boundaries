

# E001 — Artefact de build immuable par conception

Un artefact de build **ne doit jamais être modifié après sa production**.  
Toute modification implique la création d’un **nouvel artefact**.

Si l’artefact peut changer après le build,  
le pipeline **ne garantit pas** que l’on analyse ou redéploie  
le même objet que celui initialement produit.


## Risque opérationnel

- impossible d’isoler précisément un problème
- redeploy complet requis même pour un incident local
